Warm lacquered gold on mahogany — jazz pit orchestra energy.
5 instruments, each with its own physical model:

Trumpet — bright and cutting, classic upper brass harmonic stack
Flugelhorn — warmer, fewer partials, rounder attack
French Horn — rich mid-register, strong body resonance, stopped hand technique
Trombone — full low brass, fat fundamental
Tuba — deep and warm, minimal upper harmonics

Mute system — Open, Straight, Cup, Harmon, Wah, Plunger, Stopped — each reshapes the filter cutoff, Q, and notch to model the actual acoustic effect
Per-instrument knobs — Bright (harmonic emphasis), Growl (modulated distortion via sawtooth LFO), Air (breath noise layer)
Ensemble mode — click + Ensemble on any instrument to layer it alongside your primary voice with spread detuning for a section feel
Articulation — Normal, Staccato (short sharp notes), Legato (slow attack, long release), Marcato (punchy accent), Fall (pitch drop on release)
Global — Vibrato depth + rate, section spread (detuning between ensemble voices), attack/release, hall reverb, master volume
